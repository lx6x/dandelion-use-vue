# dandelion-use

## 模块结构

```
dandelion-use
├── dandelion-client                         页面
|  
├── dandelion-server                         服务
|   |   ├── dandelion-server-web             实际对外抛出服务
|   |   ├── dandelion-server-common          公共模块
└── └── └── dandelion-server-domain          系统模块
```

## 技术选型

| 框架             | 版本 | 说明       | 
|----------------|----|----------|
| Springboot     |    | 容器+MVC框架 |
| Druid          |    | 阿里数据库连接池 |
| Mybatis-Plus   |    | ORM框架    |
| Knife4j        |    | 文档生产工具   |
| Redis          |    | 分布式缓存    |
