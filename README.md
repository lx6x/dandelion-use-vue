# dandelion-use-vue

## 模块结构

```
dandelion-use
├── dandelion-client                         管理端
|  
├── dandelion-server                         服务
|   |   ├── dandelion-server-web             实际对外抛出服务
|   |   ├── dandelion-server-core            核心模块
└── └── └── dandelion-server-service         系统模块
```

## 技术选型

| 框架             | 版本 | 说明       | 
|----------------|----|----------|
| Springboot     |    | 容器+MVC框架 |
| Druid          |    | 阿里数据库连接池 |
| Mybatis-Plus   |    | ORM框架    |
| Knife4j        |    | 文档生产工具   |
| Redis          |    | 缓存    |
